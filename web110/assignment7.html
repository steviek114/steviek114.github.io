Lesson Summary: Security

In Lesson 7, we learned about Internet Security. Internet security is a type of computer security that revolves around protecting users from online threats like malware, phishing, and unauthorized access. The Internet hosts many vulnerabilities that allow threats like viruses, worms, Trojans, and other malicious software to steal data or damage systems. Some common protections against these threats are firewalls, antivirus software, and two-factor authentication.
